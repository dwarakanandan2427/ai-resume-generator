# AI Resume Generator using FLAN-T5

An AI-powered resume assistant that automatically generates professional resume content using Google's FLAN-T5 language model.

This project helps users create better resumes by generating professional summaries, suggesting relevant skills, correcting spelling mistakes, and converting job descriptions into resume-ready bullet points.

## Features

• AI generated professional summary  
• Job description to resume bullet points conversion  
• Automatic skill suggestions  
• Spell correction for resume text  
• Resume content generation pipeline  

## Tech Stack

Python  
Google FLAN-T5  
HuggingFace Transformers  
Pandas  
Scikit-learn  
Jupyter Notebook  

## How It Works

1. User inputs resume information or job description
2. FLAN-T5 processes the text
3. The model generates structured resume content
4. The system suggests skills and improves the wording

## Project Architecture

Dataset → Preprocessing → FLAN-T5 Model → Resume Generation → Output Formatting

## Applications

• Resume building tools  
• Career assistance platforms  
• AI writing assistants  
• HR automation tools
